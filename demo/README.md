## 使用

``` bash
# 安装依赖（如果生成初始化项目时已经安装完毕，无需安装）
npm install

# 开发模式 localhost:8080
npm run dev

# 构建模式
npm run build

# 构建模式 + 分析报告
npm run build --report

```
